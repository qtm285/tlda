Question Bank: Potential Outcomes

Question A: Reading a Potential Outcomes Table

Consider this population of 6 people in a randomized experiment.

\(j\) \(y_j(1)\) \(y_j(0)\) \(\tau_j\)
1 6 2 4
2 0 0 0
3 4 1 3
4 7 7 0
5 8 4 4
6 2 0 2

Part A.1

What is the average treatment effect \(\bar\tau = \frac{1}{6}\sum_{j=1}^6 \tau_j\)?

Solution

\(\bar\tau = \frac{1}{6}(4 + 0 + 3 + 0 + 4 + 2) = \frac{13}{6} \approx 2.17\)

Part A.2

Suppose we assign treatments \(W_1 \ldots W_6 = (1, 0, 0, 1, 1, 0)\). Fill in the realized outcomes \(Y_j = y_j(W_j)\) in the table below.

\(j\) \(W_j\) \(Y_j\)
1 1
2 0
3 0
4 1
5 1
6 0
Solution
\(j\) \(W_j\) \(Y_j\)
1 1 6
2 0 0
3 0 1
4 1 7
5 1 8
6 0 0

Person 1 is treated, so we observe \(y_1(1) = 6\). Person 2 is untreated, so we observe \(y_2(0) = 0\). And so on.

Part A.3

Calculate the difference-in-means estimator \(\hat\tau = \bar Y_1 - \bar Y_0\) for this treatment assignment, where \(\bar Y_1\) is the mean outcome among the treated and \(\bar Y_0\) is the mean among the untreated.

Solution

\[\hat\tau = \frac{6 + 7 + 8}{3} - \frac{0 + 1 + 0}{3} = 7 - \frac{1}{3} = \frac{20}{3} \approx 6.67\]

Part A.4

Is \(\hat\tau\) equal to \(\bar\tau\)? In one sentence, explain why they differ.

Solution

No. They differ because we’re comparing different people: the treated group happens to have higher \(y_j(0)\) values than the untreated group, which inflates our estimate.

Question B: Unbiasedness under Randomization

Using the same population as Question A, suppose we randomize by choosing 3 people uniformly at random to treat (and the other 3 are untreated).

Part B.1

There are \(\binom{6}{3} = 20\) possible treatment assignments. For the assignment \((1,1,1,0,0,0)\)—treating persons 1, 2, and 3—calculate \(\hat\tau\).

Solution

\[\hat\tau = \frac{y_1(1) + y_2(1) + y_3(1)}{3} - \frac{y_4(0) + y_5(0) + y_6(0)}{3} = \frac{6 + 0 + 4}{3} - \frac{7 + 4 + 0}{3} = \frac{10}{3} - \frac{11}{3} = -\frac{1}{3}\]

Part B.2

For the assignment \((0,0,0,1,1,1)\)—treating persons 4, 5, and 6—calculate \(\hat\tau\).

Solution

\[\hat\tau = \frac{y_4(1) + y_5(1) + y_6(1)}{3} - \frac{y_1(0) + y_2(0) + y_3(0)}{3} = \frac{7 + 8 + 2}{3} - \frac{2 + 0 + 1}{3} = \frac{17}{3} - 1 = \frac{14}{3}\]

Part B.3

If we average \(\hat\tau\) over all 20 possible assignments (each equally likely), what do we get? You don’t need to enumerate all 20—just state what the answer must be and why.

Solution

We get \(\bar\tau = 13/6 \approx 2.17\). This is because \(\hat\tau\) is an unbiased estimator of \(\bar\tau\) under randomization: \(\E[\hat\tau] = \bar\tau\). The average over all possible randomizations equals the true average treatment effect.

Question C: The Fundamental Problem

Part C.1

In the table from Question A, why can’t we calculate \(\tau_1 = y_1(1) - y_1(0)\) for person 1 in a real experiment?

Solution

Because person 1 can only receive one treatment. If they’re treated, we observe \(y_1(1)\) but not \(y_1(0)\). If they’re untreated, we observe \(y_1(0)\) but not \(y_1(1)\). We never observe both potential outcomes for any individual.

Part C.2

Given that we can’t observe individual treatment effects, explain in one or two sentences why randomization still lets us estimate the average treatment effect.

Solution

Randomization ensures that, on average, the treated and untreated groups have the same distribution of potential outcomes. So while we can’t see both \(y_j(1)\) and \(y_j(0)\) for anyone, we can compare the average outcomes across groups to estimate the average difference.

Question D: Variance of the Treatment Effect Estimator

In a randomized experiment with \(n_1\) treated and \(n_0\) untreated units, the variance of \(\hat\tau = \bar Y_1 - \bar Y_0\) is approximately \[ \Var(\hat\tau) \approx \frac{\sigma^2(1)}{n_1} + \frac{\sigma^2(0)}{n_0} \] where \(\sigma^2(w)\) is the variance of potential outcomes under treatment \(w\).

Part D.1

In an experiment with 100 treated and 100 untreated units, suppose \(\hat\sigma^2(1) = 4\) and \(\hat\sigma^2(0) = 9\) (the sample variances in each group). Calculate the estimated standard error of \(\hat\tau\).

Solution

\[\widehat{SE}(\hat\tau) = \sqrt{\frac{4}{100} + \frac{9}{100}} = \sqrt{\frac{13}{100}} = \sqrt{0.13} \approx 0.36\]

Part D.2

If \(\hat\tau = 2.5\), construct a 95% confidence interval for the average treatment effect.

Solution

\[\hat\tau \pm 1.96 \times \widehat{SE} = 2.5 \pm 1.96 \times 0.36 = 2.5 \pm 0.71 = (1.79, 3.21)\]

Question E: Two Sources of Randomness (Connects to 10aa)

In some experiments, we sample from a population and then randomize treatment within that sample.

Part E.1

Name the two sources of randomness in such an experiment.

Solution
  1. Sampling: Which units from the population end up in the experiment
  2. Randomization: Which units in the sample get assigned to treatment vs control

Part E.2

If we observe the entire population (no sampling) and only randomize treatment, which source of randomness remains?

Solution

Only randomization variability. There’s no sampling variability because we observe everyone.

Part E.3

In a large sample from a large population, which source of variability typically dominates? Why?

Solution

Randomization variability typically dominates in large samples because the sample average treatment effect is close to the population average treatment effect when \(n\) is large relative to \(m\). The standard error formulas we use (based on within-group variances) capture randomization variability.

Question F: Interpreting a Randomized Experiment (Michigan-style)

In a get-out-the-vote experiment, 250 people were randomly assigned to receive a mailer and 250 were assigned to a control group. The voting rates were:

  • Mailer group: 42% voted
  • Control group: 35% voted

Part F.1

What is the estimated average treatment effect of receiving the mailer?

Solution

\(\hat\tau = 0.42 - 0.35 = 0.07\) or 7 percentage points.

Part F.2

The estimated standard error is 0.043. Construct a 95% confidence interval for the ATE.

Solution

\(0.07 \pm 1.96 \times 0.043 = 0.07 \pm 0.084 = (-0.014, 0.154)\)

Or approximately \((-1.4\%, 15.4\%)\) in percentage points.

Part F.3

A colleague says: “The confidence interval includes zero, so the mailer had no effect.” Do you agree? Explain briefly.

Solution

Not exactly. The confidence interval tells us the range of ATEs that are consistent with our data. Zero is in this range, so we can’t rule out no effect. But neither can we rule out effects as large as 15 percentage points. The data are consistent with both “no effect” and “substantial positive effect”—we simply don’t have enough precision to tell.

Question G: Code Interpretation (Bootstrap for Treatment Effect)

We run this code on data from a randomized experiment stored in a dataframe with columns w (treatment indicator) and y (outcome).

B = 1000
tau.star = rep(NA, B)
for (b in 1:B) {
  I = sample(1:n, n, replace = TRUE)
  w.star = w[I]
  y.star = y[I]
  tau.star[b] = mean(y.star[w.star == 1]) - mean(y.star[w.star == 0])
}

Part G.1

What does tau.star contain after this code runs?

Solution

tau.star contains 1000 bootstrap estimates of the treatment effect. Each is the difference in mean outcomes between treated and control in a bootstrap sample.

Part G.2

How would you use tau.star to construct a 95% confidence interval for the ATE?

Solution

Calculate the point estimate tau.hat = mean(y[w==1]) - mean(y[w==0]) and the bootstrap standard error se = sd(tau.star). Then the 95% CI is tau.hat ± 1.96 * se.

Part G.3

What does sd(tau.star) estimate?

Solution

It estimates the standard error of \(\hat\tau\)—the standard deviation of the sampling distribution of the difference-in-means estimator.
