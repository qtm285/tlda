How it works

The server renders this .md file with markdown-it + KaTeX and serves it as an HTML page inside a TLDraw canvas. No LaTeX installation needed.

What you can do

  • Draw on the document with the pen or highlighter
  • Drop math notes — press m, click to place, type $\LaTeX$
  • Ping to get an agent’s attention (bottom-right button)
  • Everything syncs in real time across all viewers

A theorem

Theorem (Cauchy-Schwarz). For any vectors u,vu, vu,v in an inner product space,

∣⟨u,v⟩∣2≤⟨u,u⟩⋅⟨v,v⟩.|\langle u, v \rangle|^2 \leq \langle u, u \rangle \cdot \langle v, v \rangle.∣⟨u,v⟩∣2≤⟨u,u⟩⋅⟨v,v⟩.

Equality holds if and only if uuu and vvv are linearly dependent.
